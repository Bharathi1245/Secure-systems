# Project Setup

## Requirements

Make sure you have the following installed:

- **Python** (version 3.8 or higher recommended)
- **pip** (Python package manager)

## Installation

Install the required dependencies:

```bash
python -m pip install pillow
```

Run:
```bash
py task1.py
```
